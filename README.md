# Logistics-Carbon-Emissions-Analysis
A Power BI project analyzing logistics emissions, identifying key drivers, and providing data-driven recommendations for sustainable delivery operations.

# About the Project

This project analyzes a logistics dataset of 12,000 deliveries to understand what drives carbon emissions in delivery operations and how they can be reduced.

Using Excel and Power BI, I built dashboards to track performance, identify emission hotspots, and simulate the impact of fleet electrification on sustainability goals.

# Objectives
Identify high-emission routes and locations
Analyze the impact of traffic and distance on emissions
Compare efficiency across vehicle types
Estimate emission reduction using eco-friendly vehicles

# Key Insights
Avg delivery distance: 103 km | Avg emissions: 72 kgCO₂e
Inter-city routes generate the highest emissions
Traffic congestion significantly increases emissions
Electric vehicles are more efficient, while trucks emit the most
Only 43% of deliveries are eco-friendly

# Impact
Replacing 50% of fossil-fuel vehicles can reduce emissions by ~49%
Highlights clear opportunities for route optimization and green logistics

# Recommendations
Increase adoption of electric vehicles & cargo bikes
Optimize long-distance deliveries
Schedule deliveries during low-traffic periods
Focus sustainability efforts on high-emission locations

# Tools Used
Excel – Data cleaning
Power BI – Dashboard & visualization
